## Dependencies

- Node.js
- body-parser
- chance
- express
- http-server
- md5
- timeago.js

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run `npm install nodemon`, then start the server using `npm start`. (or run the development web server using the `npm run local` command.)
